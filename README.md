# منصة كشف | Kashf Platform

منصة تحليل بيانات لاكتشاف التعثر في الخدمات الرقمية وتحليل رحلة المستخدم داخل الخدمات الحكومية.

تعتمد المنصة على تحليل بيانات الجلسات واكتشاف الأنماط غير الطبيعية (Anomalies) ونقاط الانسحاب لتحسين تجربة المستخدم وجودة الخدمات الرقمية.

---

## فكرة المشروع

الكثير من الخدمات الرقمية تواجه مشاكل مثل:

- بطء الاستجابة
- أخطاء النظام
- انسحاب المستخدم قبل إكمال الخدمة

منصة **كشف** تقوم بتحليل هذه البيانات واكتشاف:

- أماكن التعثر في رحلة المستخدم
- الخدمات الأكثر حدوثًا للأخطاء
- نقاط الانسحاب في النماذج
- الأنماط غير الطبيعية في الأداء

---

## أهم المميزات

- لوحة مؤشرات تفاعلية (Dashboard)
- تحليل رحلة المستخدم (User Journey)
- اكتشاف الشذوذ (Anomaly Detection)
- تحليل أنماط التعثر باستخدام Clustering
- تحديد نقاط الانسحاب في الخدمات
- نظام تنبيهات عند تكرار المشكلة

## التحليلات داخل المنصة

تتضمن المنصة عدة تحليلات رئيسية:

### نظرة عامة
تحليل المؤشرات الرئيسية مثل:
- معدل الأخطاء
- معدل الانسحاب
- زمن الاستجابة

### تحليل نقاط الانسحاب
تحديد المرحلة التي يترك فيها المستخدم الخدمة.

### كشف الشذوذ
اكتشاف الارتفاع غير الطبيعي في الأخطاء أو البطء.

### أنماط التعثر
تجميع الجلسات المتشابهة لمعرفة أنماط المشاكل.

### نظام الإنذارات
إظهار المشاكل التي تتكرر بشكل كبير.
